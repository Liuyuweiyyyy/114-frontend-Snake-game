# 🐍 貪食蛇

單人操作貪食蛇遊戲 — 玩家蛇對抗機器敵人蛇，並可使用技能消滅敵人。

## 設計理念

這版加入了「技能成長」與「敵蛇搶果子」的機制，讓原本單純追求高分的貪食蛇，變成更具有策略性的遊戲。最初只有加入敵方蛇時，雖然牠們速度較慢，但因為能快速追蹤果子，仍然會對玩家造成很大的壓力，因此我加入技能系統來平衡玩法。

玩家在吃果子升級時，可以選擇不同技能來限制敵蛇行動，例如守護蛇、閃電與冰凍等能力。玩家除了控制移動路線，也需要觀察敵蛇位置、技能冷卻時間，以及技能施放時機，增加更多即時判斷與策略思考。

我希望玩完後，能感受到這不只是傳統 SnakeGame，而是一款結合技能與對抗元素的進化版貪食蛇。

---

## 遊戲規則

- **玩家蛇**：方向鍵控制，綠色
- **敵人蛇（AI）**：自動追蹤食物，淡紅色，長度固定 3, 每4級多一隻，最多5隻
- **食物**：吃到 +20 分
- **加速**：每吃 5 顆升一等級並加速（初始 150ms → 最快 60ms）
- **穿牆**：蛇碰到邊緣會從另一邊穿出
- **Game Over**：玩家蛇撞到自己即結束

## 操作

|       按鍵    |           功能        |
|---------------|----------------------|
|      方向鍵    |     控制玩家蛇方向    |
|      空白鍵    |   召喚守護蛇技能      |
|        R      |       暫停 / 繼續     |
|      任意鍵    |   遊戲畫面開始遊戲    |
| 空白鍵 | Game Over 後重新開始 |

## 技能系統

### 技能升級
- 玩家每次升級（每吃 5 顆）時可選擇技能卡牌
- 未擁有的技能可以解鎖，未滿級的技能可以升級
- 選項不足時會出現「分數+100」填充

### 守護蛇

| 等級 | 效果 |
|------|------|
| Lv.1 | 在玩家右側生成守護蛇，以 1.5 倍速追蹤敵蛇，撞到時眩暈 2 秒 |
| Lv.2 | 速度提升至 2.0 倍 |
| Lv.3 | 召喚兩隻蛇（左一右一），眩暈時再次撞擊則眩暈完停頓 3 秒 |

- 撞到敵蛇後立即消失
- 技能冷卻 5 秒

### 閃電

| 等級 | 範圍 | 敵蛇眩暈 | 玩家加速 |
|------|------|----------|----------|
| Lv.1 | 4×4 | 3 秒 | ×1.2 倍 |
| Lv.2 | 4×4 | 4 秒 | ×1.5 倍 |
| Lv.3 | 5×5 | 4 秒 | ×1.5 倍 |

- 預警 2 秒後閃電
- 擊中蛇身任意部位都有效
- 玩家加速時蛇身會泛黃發光
- 冷卻 5 秒，自動釋放

### 冰凍

| 等級 | 範圍 | 範圍內減速 | 離開後減速 |
|------|------|-----------|------------|
| Lv.1 | 3×3 | 30% | 20%（持續 3 秒）|
| Lv.2 | 3×3 | 50% | 30%（持續 3 秒）|
| Lv.3 | 5×5 | 50% | 30%（持續 3 秒）|

- 範圍跟隨玩家蛇頭移動
- 敵蛇身體任意部位在範圍內都會減速
- 被減速的敵蛇會顯示冰藍色邊框
- 冷卻 5 秒，自動釋放

### 天譴之龍

| 等級 | 條件 | 效果 |
|------|------|------|
| Lv.1 | 守護蛇 Lv.3 + 閃電 Lv.3 | 召喚6隻天譴之龍（金色），速度 2.0 倍，持續 3 秒 |

- 6隻龍出現在玩家蛇頭左3右3，各追蹤最近的敵蛇
- 撞擊效果依次循環：
  - 第1次撞擊：眩暈 2 秒
  - 第2次撞擊：停頓 3 秒
  - 第3次撞擊：連續雷擊（以該敵蛇為中心 5×5 範圍，周圍敵蛇眩暈 1 秒 + 停頓 1 秒，玩家加速 2 秒）
  - 之後繼續循環
- 連續雷擊有 2 秒預警（黃色），轟炸時白色閃爍覆蓋在上層
- 冷卻 5 秒，自動釋放
- **解鎖後影響**：守護蛇和閃電技能隱藏，按空白鍵無法召喚守護蛇

### 極寒領主
| 等級 | 條件 | 效果 |
|------|------|------|
| Lv.1 | 守護蛇 Lv.3 + 冰凍 Lv.3 | 5×5 減速範圍 + 冰蛇圓周運動 + 冰路 |
- 減速範圍：淡藍色半透明，跟隨玩家蛇頭
- 冰蛇：深藍色，圍繞玩家蛇頭逆時針旋轉，速度與玩家相同
- 冰路：天藍色，冰蛇移動軌跡，每格持續3秒
- 敵蛇在減速範圍內：減速50%
- 敵蛇撞到冰蛇頭：冰凍5秒（無法移動）
- 敵蛇在冰路上造成打滑(無法改變方向)
- 持續3秒，冷卻5秒，自動釋放
- **解鎖後影響**：守護蛇和冰凍技能隱藏，按空白鍵無法召喚守護蛇

### 凜冬雷暴
| 等級 | 條件 | 效果 |
|------|------|------|
| Lv.1 | 冰凍 Lv.3 + 閃電 Lv.3 | 5×5 風暴範圍 + 風暴蛇隨機移動 + 吸附 + 混亂 |
- 風暴範圍：淡紫色半透明 5×5 區域，風暴蛇周圍
- 風暴蛇：淡紫色，長度 5 格，速度 2.0 倍
- 移動邏輯：
  - 有敵蛇在風暴範圍外時 → 追蹤最近的範圍外敵蛇
  - 所有敵蛇都在風暴範圍內時 → 隨機亂走
- 吸附效果：5×5 範圍內的敵蛇會被強制繞著風暴中心逆時針打轉
- 混亂效果：風暴結束時，5×5 範圍內的敵蛇進入混亂狀態 3 秒（隨機變向）
- 持續 3 秒，冷卻 5 秒，自動釋放
- 解鎖後影響：守護蛇、冰凍、閃電技能隱藏，按空白鍵無法召喚守護蛇
