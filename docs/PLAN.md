# 貪食蛇遊戲開發記錄

## 專案概述

- **技術棧**：Vue 3 + TypeScript + Vite
- **遊戲範圍**：20×20 網格
- **主要元件**：
  - `App.vue` - 主應用（含主菜單）
  - `SnakeGame.vue` - 主要遊戲
  - `TutorialGame.vue` - 教學關卡
  - `composables/useLeaderboard.ts` - 排行榜共用邏輯

---

## 遊戲狀態

| 狀態 | 說明 |
|------|------|
| `idle` | 初始狀態，等待玩家開始 |
| `playing` | 遊戲進行中 |
| `paused` | 玩家主動暫停 |
| `levelup` | 升級選擇技能卡牌 |
| `gameover` | 遊戲結束 |

---

## 核心變數

| `snake` | `Ref<Position[]>` | 玩家蛇身座標（頭在 index 0） |

| 函式 | 說明 |
|------|------|
| `moveSnake()` | 通用移動函式（玩家/AI/守護蛇共用） |
| `getAIDirection()` | AI 計算路徑（避開障礙物） |

## 遊戲機制

### 穿牆
- 碰到牆壁從另一邊穿出
- 座標計算：`((x % GRID_WIDTH) + GRID_WIDTH) % GRID_WIDTH`

### 等級系統
- 每吃 5 顆升一級
- 速度遞增：基礎 150ms，每級減少 10ms，最快 60ms
- 升級時顯示技能卡牌選擇

### 分數
- 玩家吃食物：+20 分
- AI 吃食物：-10 分

---

## 控制系統

| 按鍵 | 功能 |
|------|------|
| 方向鍵 | 移動 / 開始遊戲 |
| R | 暫停/繼續（需按兩次） |
| 空白鍵 | 召喚守護蛇 |
| 1/2/3 | 選擇技能 |
| 兩次空白鍵 | 遊戲結束後重新開始 |
| 兩次 R 鍵 | 跳過技能選擇 |

---

## AI 蛇系統

### 生成規則
- 數量：`1 + floor((玩家等級 - 1) / 3)`
- 初始：1 條
- 長度：固定 3

### 負面效果（每條獨立）
| 效果 | 說明 |
|------|------|
| `stunned` | 眩暈（1秒/4秒） |
| `paused` | 停頓（3秒） |
| `slowed` | 減速（50%/70%） |

---

## 技能系統

### 守護蛇
| 等級 | 速度 | 效果 |
|------|------|------|
| Lv.1 | 1.5x | 眩暈 1 秒 |
| Lv.2 | 2.0x | 眩暈 1 秒 |
| Lv.3 | 2.0x | 雙蛇 + 眩暈後停頓 3 秒 |

### 閃電
| 等級 | 範圍 | 預警 | AI 眩暈 | 玩家加速 |
|------|------|------|----------|----------|
| Lv.1 | 4×4 | 2秒 | 3秒 | 1.2x |
| Lv.2 | 4×4 | 2秒 | 4秒 | 1.5x |
| Lv.3 | 5×5 | 2秒 | 4秒 | 1.5x |

### 冰凍
| 等級 | 範圍 | 範圍內敵人 | 離開後敵人 |
|------|------|--------|--------|
| Lv.1 | 3×3 | 70% 速度 | 80% 持續 3 秒 |
| Lv.2 | 3×3 | 50% 速度 | 70% 持續 3 秒 |
| Lv.3 | 5×5 | 50% 速度 | 70% 持續 3 秒 |

## 主菜單

- 三個按鈕：開始遊戲、排行榜、教學關卡（寬度 220px）
- 排行榜使用 `useLeaderboard` composable 共用資料

---

## 教學關卡

### 區域劃分
- 玩家果子：(12, 16)
- AI 果子：(10, 5)

### 流程
1. 玩家吃果子 → 出現 AI 蛇
2. AI 吃果子 → 扣 10 分
3. 提示按空白召喚守護蛇
4. 守護蛇撞 AI → AI 眩暈 → 2 秒後彈窗

### 完成彈窗
- 標題：教學完成！
- 按鈕：返回主畫面、開始遊戲（200px）

## Bug 修復記錄

- 每次重新開始會加速 → 清除舊計時器
- 結束畫面輸入名字時按鍵觸發開始 → 限定方向鍵/空白鍵
- 暫停時冷卻繼續計時 → 清除冷卻計時器
- 按 LEFT 開始會立即撞死 → 保持向右移動
- 升級時顯示暫停畫面 → 新增 `levelup` 狀態
- 重新開始有殘留特效 → `resetGame()` 清除所有狀態
