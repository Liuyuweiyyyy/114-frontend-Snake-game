# 貪食蛇遊戲開發記錄

## 專案背景
- 基底專案：Vue 3 + TypeScript + Vite
- 遊戲範圍：20×20 網格
- 開發方式：三步驟逐步實作

---

## Step 1 — HTML 架構

### 用戶需求
建立遊戲的 HTML 架構。

### 實作內容
- 在 `src/components/SnakeGame.vue` 建立貪食蛇元件
- 使用 CSS Grid 產生 20×20 網格（400 個 `.cell` div，用 `v-for` 產生）
- 每個格子預留 snake / head / food 三種動態 class
- 包含標題「貪食蛇」、計分板、開始按鈕、遊戲結束 overlay
- 修改 `App.vue` 只顯示 `<SnakeGame />`

---

## Step 2 — 變數設計

### 用戶需求
定義遊戲所需的核心變數。

### 實作內容
在 `<script setup lang="ts">` 中宣告：

| 變數 | 類型 | 說明 |
|------|------|------|
| `gridSize` | `number` | 網格大小 (20) |
| `snake` | `Ref<Position[]>` | 蛇身座標陣列（頭在第 0 項） |
| `food` | `Ref<Position>` | 食物座標 |
| `direction` | `Ref<Direction>` | 當前方向 (`'UP'` / `'DOWN'` / `'LEFT'` / `'RIGHT'`) |
| `score` | `Ref<number>` | 分數 |
| `gameStatus` | `Ref<GameStatus>` | 遊戲狀態 (`'idle'` / `'playing'` / `'paused'` / `'gameover'`) |
| `gameTimer` | `ReturnType<typeof setInterval> \| null` | setInterval 計時器 ID |
| `pendingDirection` | `Direction \| null` | 暫存的下個方向（防止 180° 回頭） |

### 補充說明
- `interface Position` 定義座標結構 `{ x: number; y: number }`
- `type Direction` 為聯合型別（類似 C++ enum）
- `type GameStatus` 定義四種遊戲狀態

---

## Step 3 — Function 實作

### 用戶需求
完成所有遊戲邏輯函式。

### 實作內容
所有 function 寫在 `<script setup lang="ts">` 中：

| 函式 | 說明 |
|------|------|
| `getCellType(index)` | 根據格子索引回傳 `'head'` / `'snake'` / `'food'` / `null` |
| `generateFood()` | 隨機生成食物（避開蛇身） |
| `checkSelfCollision(head)` | 檢查蛇頭是否撞到身體 |
| `move()` | 每 tick 移動蛇：計算新蛇頭、穿牆、碰撞判斷、吃食物 |
| `changeDirection(newDir)` | 防 180° 回頭的方向切換 |
| `gameLoop()` | 啟動 setInterval（每 150ms 呼叫 move） |
| `startGame()` | 重置所有狀態並開始遊戲 |
| `togglePause()` | 暫停／繼續切換 |
| `handleKeydown(e)` | 鍵盤事件處理 |
| `onMounted / onUnmounted` | 註冊／移除鍵盤監聽 |

### 涉及修改
- Template 中 `:class` 改為 `:class="getCellType(i)"`
- 分數改為動態綁定 `{{ score }}`
- 按鈕綁定 `@click` 事件
- Overlay 加上 `:class="{ visible: ... }"` 控制顯示

## 後續調整記錄

---

### 遊戲機制

#### 穿牆機制
- **需求**：碰到牆壁不死亡，改為從另一邊穿出來
- **做法**：
  - `move()` 中計算新蛇頭後，用 `((x % gridSize) + gridSize) % gridSize` 進行座標繞圈
  - `checkSelfCollision` 只檢查是否撞到自己（移除牆壁碰撞檢查）

#### 加速與等級系統
- **需求**：每吃食物加速，等級提升
- **做法**：
  - 新增 `eatenCount` 追蹤已吃食物數
  - 每吃 5 顆升一級（基礎間隔 150ms，每級縮短 10ms，最快 60ms）
  - 每吃一顆食物 +20 分
  - 等級卡片顯示 `Lv.X` 及進度條（0-100%）

---

### 控制系統

#### 開始遊戲
- **調整**：移除開始按鈕，改為任意鍵開始
- **做法**：方向鍵也可開始遊戲並設定初始方向

#### 暫停功能
- **調整**：暫停改為 R 鍵（空白鍵預留給技能）
- **做法**：`e.key === 'r' || e.key === 'R'` 時呼叫 `togglePause()`

#### 重新開始
- **調整**：結束畫面重新開始改為特定按鍵
- **做法**：gameover 狀態時，只有方向鍵或空白鍵才觸發儲存並重新開始

#### 狀態管理
- **問題**：`'idle'` 同時代表初始狀態和暫停狀態
- **解決**：新增 `'paused'` 狀態，`togglePause()` 在 `'playing'` ↔ `'paused'` 之間切換

---

### UI / 顯示

#### 配色改為淡藍色系
- 背景：淡藍色漸層 `#e8f4fd → #b3d9f2`
- 空格：`#d4eaf7`、蛇身：`#27ae60`、蛇頭：`#1a7a3a`、食物：`#e74c3c`

#### 右側面板
- 新增 `.side-info` 固定在遊戲區右側
- 分數卡片（藍色漸層）、等級卡片（綠色漸層 + 進度條）

#### 排行榜
- `localStorage` 儲存，最多 10 筆，按分數降冪排列
- 支援輸入名稱儲存（留空為「匿名」）
- 提供「清空」按鈕

---

### AI 蛇（敵人）

- 新增 `aiSnake`（淡紅色 `#f5b7b1`，頭部 `#e74c3c`），長度固定 3
- AI 邏輯：計算最近的食物位置並移動（加入障礙物檢測）
- AI 速度：0.6 倍（較慢），使用獨立定時器
- AI 吃到食物：扣 10 分（最低 0 分）
- 食物生成時避開兩條蛇，吃到不變長（`grow: false`）
- 暫停/遊戲結束時清除 AI 定時器

---

### 遊戲模式演變

#### 雙蛇模式（已廢除）
- 遊戲區分割為左右兩塊 10×20 螢幕
- 鏡像操控模式：左蛇上下同向、左右反向
- 食物必須兩邊都吃完才重生

#### 改回單蛇模式
- 網格改回 20×20
- 移除雙蛇相關變數和鏡像操控邏輯

---

### Bug 修復

#### 每次重新開始會加速
- **原因**：未清除舊的 `setInterval`
- **解決**：在 `gameLoop()` 和 `startGame()` 開頭加上 `clearInterval`

#### 結束畫面輸入名字問題
- **問題**：按鍵會觸發儲存並重新開始
- **解決**：只有方向鍵或空白鍵才會觸發

#### 暫停時冷卻繼續計時
- **問題**：按 R 暫停時，守護蛇和閃電冷卻還在繼續
- **解決**：暫停時清除所有冷卻計時器，繼續時重新啟動

---

### 程式碼重構

#### 守護蛇冷卻變數
- `defenderCooldown` 從 `boolean` 改為 `number`（剩餘毫秒）
- 冷卻時每秒遞減，UI 綁定計算剩餘百分比

### 技能系統

#### 技能等級數據結構
| 技能 | 初始等級 | 最高等級 | 數據結構 |
|------|----------|----------|----------|
| 守護蛇 | Lv.1（已解鎖）| Lv.3 | `{ level: 1, unlocked: true }` |
| 閃電 | Lv.0（未解鎖）| Lv.3 | `{ level: 0, unlocked: false, cooldown: 0 }` |
| 冰凍 | Lv.0（未解鎖）| Lv.3 | `{ level: 0, unlocked: false, cooldown: 0 }` |

#### 升級卡牌系統
- **觸發時機**：玩家每次升級（每吃 5 顆）時暫停遊戲顯示卡牌
- **顯示內容**：未擁有技能或未滿級技能
- **填充選項**：分數+100（不足 3 個時填充）

#### 守護蛇技能
| 等級 | 效果 |
|------|------|
| Lv.1 | 一隻蛇在玩家右側以玩家 1.5 倍速追蹤 AI 蛇，眩暈 2 秒 |
| Lv.2 | 速度強化成 2.0 倍 |
| Lv.3 | 召喚兩隻蛇（左一右一），眩暈時再次撞擊則眩暈完停頓 3 秒 |
- **撞到消失**：撞到 AI 蛇後觸發效果並立即消失

#### 閃電技能
| 等級 | 範圍 | 預警 | AI 眩暈 | 玩家加速 |
|------|------|------|----------|----------|
| Lv.1 | 4×4 | 2 秒 | 3 秒 | 1.2 倍 |
| Lv.2 | 4×4 | 2 秒 | 4 秒 | 1.5 倍 |
| Lv.3 | 5×5 | 2 秒 | 4 秒 | 1.5 倍 |

- **擊中判定**：檢查蛇身任意部位，不只是頭部
- **玩家加速特效**：蛇頭和蛇身外圍泛黃發光
- **自動釋放**：冷卻完畢後自動觸發
- **冷卻時間**：5 秒

#### 冰凍技能
| 等級 | 範圍 | 範圍內減速 | 離開後減速 |
|------|------|-----------|------------|
| Lv.1 | 3×3 | 30% | 額外 20%（持續 3 秒）|
| Lv.2 | 3×3 | 50% | 額外 30%（持續 3 秒）|
| Lv.3 | 5×5 | 50% | 額外 30%（持續 3 秒）|

- **範圍位置**：以玩家蛇頭為中心
- **視覺效果**：
  - 冰凍範圍跟隨玩家蛇頭移動
  - 冰凍範圍在所有蛇和食物之下（最下層）
  - AI 蛇被減速時顯示冰藍色邊框發光
- **減速實現**：每次移動時重新設置定時器間隔，動態調整速度
- **效果獨立**：AI 蛇身體任意部位在範圍內也會減速

### AI 蛇系統

#### 多條 AI 蛇
- **生成規則**：玩家每吃 15 顆食物（玩家 Lv.3）多一條 AI 蛇
- **數量公式**：`Math.floor((玩家等級 - 1) / 3) + 1`
- **初始數量**：1 條
- **長度**：固定 3
- **移動**：每條蛇獨立移動計時器

#### 負面效果獨立
每條 AI 蛇有獨立的眩暈、停頓、減速狀態，`interface AISnake` 新增屬性：

| 屬性 | 類型 | 說明 |
|------|------|------|
| `stunned` | `boolean` | 眩暈狀態 |
| `paused` | `boolean` | 停頓狀態 |
| `pauseStepCount` | `number` | 停頓步數計數 |
| `stunTimer` | `setTimeout` | 眩暈計時器 |
| `pauseTimer` | `setTimeout` | 停頓計時器 |
| `hitDuringStun` | `boolean` | 眩暈期間被撞擊標記 |
| `slowed` | `boolean` | 減速狀態 |
| `slowMultiplier` | `number` | 減速倍率 |
| `slowTimer` | `setTimeout` | 減速計時器 |

- **守護蛇追蹤**：追蹤最近的一條 AI 蛇
- **眩暈時減速**：AI 蛇眩暈時冰凍仍能減速
