# 貪食蛇遊戲開發記錄

## 專案背景
- 基底專案：Vue 3 + TypeScript + Vite
- 遊戲範圍：20×20 網格
- 開發方式：三步驟逐步實作

---

## Step 1 — HTML 架構

### 用戶需求
建立遊戲的 HTML 架構。

### 實作內容
- 在 `src/components/SnakeGame.vue` 建立貪食蛇元件
- 使用 CSS Grid 產生 20×20 網格（400 個 `.cell` div，用 `v-for` 產生）
- 每個格子預留 snake / head / food 三種動態 class
- 包含標題「貪食蛇」、計分板、開始按鈕、遊戲結束 overlay
- 修改 `App.vue` 只顯示 `<SnakeGame />`

---

## Step 2 — 變數設計

### 用戶需求
定義遊戲所需的核心變數。

### 實作內容
在 `<script setup lang="ts">` 中宣告：

| 變數 | 類型 | 說明 |
|------|------|------|
| `gridSize` | `number` | 網格大小 (20) |
| `snake` | `Ref<Position[]>` | 蛇身座標陣列（頭在第 0 項） |
| `food` | `Ref<Position>` | 食物座標 |
| `direction` | `Ref<Direction>` | 當前方向 (`'UP'` / `'DOWN'` / `'LEFT'` / `'RIGHT'`) |
| `score` | `Ref<number>` | 分數 |
| `gameStatus` | `Ref<GameStatus>` | 遊戲狀態 (`'idle'` / `'playing'` / `'paused'` / `'gameover'`) |
| `gameTimer` | `ReturnType<typeof setInterval> \| null` | setInterval 計時器 ID |
| `pendingDirection` | `Direction \| null` | 暫存的下個方向（防止 180° 回頭） |

### 補充說明
- `interface Position` 定義座標結構 `{ x: number; y: number }`
- `type Direction` 為聯合型別（類似 C++ enum）
- `type GameStatus` 定義四種遊戲狀態

---

## Step 3 — Function 實作

### 用戶需求
完成所有遊戲邏輯函式。

### 實作內容
所有 function 寫在 `<script setup lang="ts">` 中：

| 函式 | 說明 |
|------|------|
| `getCellType(index)` | 根據格子索引回傳 `'head'` / `'snake'` / `'food'` / `null` |
| `generateFood()` | 隨機生成食物（避開蛇身） |
| `checkSelfCollision(head)` | 檢查蛇頭是否撞到身體 |
| `move()` | 每 tick 移動蛇：計算新蛇頭、穿牆、碰撞判斷、吃食物 |
| `changeDirection(newDir)` | 防 180° 回頭的方向切換 |
| `gameLoop()` | 啟動 setInterval（每 150ms 呼叫 move） |
| `startGame()` | 重置所有狀態並開始遊戲 |
| `togglePause()` | 暫停／繼續切換 |
| `handleKeydown(e)` | 鍵盤事件處理 |
| `onMounted / onUnmounted` | 註冊／移除鍵盤監聽 |

### 涉及修改
- Template 中 `:class` 改為 `:class="getCellType(i)"`
- 分數改為動態綁定 `{{ score }}`
- 按鈕綁定 `@click` 事件
- Overlay 加上 `:class="{ visible: ... }"` 控制顯示

## 後續調整記錄

---

### 遊戲機制

#### 穿牆機制
- **需求**：碰到牆壁不死亡，改為從另一邊穿出來
- **做法**：
  - `move()` 中計算新蛇頭後，用 `((x % gridSize) + gridSize) % gridSize` 進行座標繞圈
  - `checkSelfCollision` 只檢查是否撞到自己（移除牆壁碰撞檢查）

#### 加速與等級系統
- **需求**：每吃食物加速，等級提升
- **做法**：
  - 新增 `eatenCount` 追蹤已吃食物數
  - 每吃 5 顆升一級（基礎間隔 150ms，每級縮短 10ms，最快 60ms）
  - 每吃一顆食物 +20 分
  - 等級卡片顯示 `Lv.X` 及進度條（0-100%）

---

### 控制系統

#### 開始遊戲
- **調整**：移除開始按鈕，改為任意鍵開始
- **做法**：方向鍵也可開始遊戲並設定初始方向

#### 暫停功能
- **調整**：暫停改為 R 鍵（空白鍵預留給技能）
- **做法**：`e.key === 'r' || e.key === 'R'` 時呼叫 `togglePause()`

#### 重新開始
- **調整**：結束畫面重新開始改為特定按鍵
- **做法**：gameover 狀態時，只有方向鍵或空白鍵才觸發儲存並重新開始

#### 狀態管理
- **問題**：`'idle'` 同時代表初始狀態和暫停狀態
- **解決**：新增 `'paused'` 狀態，`togglePause()` 在 `'playing'` ↔ `'paused'` 之間切換

#### 配色改為淡藍色系
- 背景：淡藍色漸層 `#e8f4fd → #b3d9f2`
- 空格：`#d4eaf7`、蛇身：`#27ae60`、蛇頭：`#1a7a3a`、食物：`#e74c3c`

#### 右側面板

#### 排行榜

- 支援輸入名稱儲存（留空為「匿名」）

---

### AI 蛇（敵人）

- 新增 `aiSnake`（淡紅色 `#f5b7b1`，頭部 `#e74c3c`），長度固定 3
- AI 邏輯：計算最近的食物位置並移動（加入障礙物檢測）
- AI 速度：0.6 倍（較慢），使用獨立定時器
- AI 吃到食物：扣 10 分（最低 0 分）
- 食物生成時避開兩條蛇，吃到不變長（`grow: false`）
- 暫停/遊戲結束時清除 AI 定時器

---

### 技能系統

#### 守護蛇技能
- **觸發**：按空白鍵
- **效果**：在玩家蛇右側生成淡青綠色守護蛇（長度 1）
- **行為**：100% 追蹤 AI 蛇，速度為玩家蛇的 1.5 倍
- **持續**：3 秒後消失
- **攻擊**：撞到 AI 蛇時，AI 眩暈 2 秒
- **冷卻**：5 秒，使用時鐘圓環 UI 顯示進度

#### 闪电技能（第二個技能）
- **解鎖**：玩家達到 Lv.2
- **範圍**：4×4 格，隨機位置
- **時序**：預警 2 秒（淡黃色 `#ffeaa7`）→ 閃電特效（白色 `#fff` + 閃爍動畫）→ 效果結算
- **效果**：
  - AI 在範圍內：眩暈 3 秒
  - 玩家在範圍內：速度 ×1.2，持續 3 秒
- **冷卻**：5 秒，自動釋放（冷卻完畢後自動觸發）

#### 技能 UI
- 遊戲區域右下角顯示技能圖標（時鐘樣式冷卻圓環）
- 守護蛇：🐍，闪电：⚡（Lv.2 後出現）
- 冷卻時透明度變暗，結束時變亮
- 暫停時所有冷卻停止，繼續時重新計時

### Bug 修復

#### 每次重新開始會加速
- **原因**：未清除舊的 `setInterval`
- **解決**：在 `gameLoop()` 和 `startGame()` 開頭加上 `clearInterval`

#### 結束畫面輸入名字問題
- **問題**：按鍵會觸發儲存並重新開始
- **解決**：只有方向鍵或空白鍵才會觸發

#### 暫停時冷卻繼續計時
- **問題**：按 R 暫停時，守護蛇和閃電冷卻還在繼續
- **解決**：暫停時清除所有冷卻計時器，繼續時重新啟動
